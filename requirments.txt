Поиск																										
		Введите станцию отправления -> введите станцию назначения -> Выберете время -> Добавить в избранное
																					|> Подписаться -> Ввести дату -> Предложить использовать последнюю (DATE) -> Ввести пожелания*
																								   |> Ввести диапазон дат -> Предложить использовать последнюю (DATE) -> Ввести пожелания*
																					|> Отписаться  -> Ввести дату
																								   |> Ввести диапазон дат
Избранное
		Подписаться -> Ввести дату -> Предложить использовать последнюю (DATE) -> Ввести пожелания*
        		    |> Ввести диапазон дат -> Предложить использовать последнюю (DATE) -> Ввести пожелания*
        Отписаться  -> Ввести дату
                    |> Ввести диапазон дат
		Удалить из избранного			
Статус					
		Выводит информацию о максимально возможном количестве подписок 
							 максимальном периоде обновления 
							 команды модератора, если пользователь модератор

Обратная связь
		Отправить сообщение всем модераторам*
		
Команды модератора:		
		Получить пользователей за последний день/неделю/час/...
		Изменить пользователю ограничения
		Заблокировать пользователя
		Разблокировать пользователя
		Просмотреть сообщения* -> Просмотреть текст сообщения -> Ответить на сообщение
		Отправить сообщение пользователю*

Пожелания: Цена, время, верхнее, нижнее, боковое, приоритет (период обновления), уведомить если остается мало мест

Отправить сообщение всем модераторам: модераторы получают уведомление
Отправить сообщение пользователю: пользователь получает уведомление. Поддерживается только текст.
Просмотреть сообщения: хранятся все сообщения списком(Отправитель - Получатель - Дата - Прочитано/непрочитано). 
Сообщение приходит пользователю в чат вместе с сообщением на которое получен ответ и сохраняется только в чате.

На некорректные команды ответ, что команды не найдена
Авторизация по tgid
